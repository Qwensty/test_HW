Характеристика типов кожи

  • Для ЖИРНОГО ТИПА КОЖИ характерно: плотная кожа с широкими порами, жирный блеск на всех участках кожи лица, наличие открытых (черные точки) и закрытых комедонов, воспалений.
  • Для СУХОГО ТИПА КОЖИ характерно: чувство стянутости (особенно после умывания), присутствует шелушение, кожа тонкая и матовая с незаметными порами, без воспалений.
  • Для НОРМАЛЬНОГО ТИПА КОЖИ характерно: небольшой жирный блеск в Т- зоне, тонкие поры, воспаления могут появляться, но очень редко.
